# MiniMax Audio Studio

MiniMax 音视频创作工作台，本地网页应用，支持音乐生成、翻唱创作、音色克隆、语音合成和图片生成。

## 功能概览

| 模块 | 说明 |
|------|------|
| 音乐生成 | 输入描述和歌词，生成完整歌曲 |
| AI 生成歌词 | 输入创作指令，AI 自动生成歌词 |
| 翻唱生成 | 上传原唱或提供 URL，转换为目标风格翻唱 |
| 音色克隆 | 上传音频样本，克隆自定义音色 |
| 语音合成 | 文本转语音，支持多语言和丰富音色 |
| 图片生成 | 文生图 / 图生图，批量生成，实时预览 |
| 记录管理 | 本地 SQLite 存储所有生成文件，在线播放和下载 |

## 快速开始

```bash
npm install
npm run dev
```

访问 `http://localhost:5173`，进入「设置」页面保存 MiniMax API Key 后即可使用。

## 页面功能

### 音乐生成

- 选择 `music-2.6-free`（免费）或 `music-2.6` 模型
- 填写歌曲描述（风格、情绪、适用场景）和歌词
- 支持纯音乐模式，设置采样率和比特率
- AI 生成歌词：填写创作指令（主题、风格），点击生成后自动填入歌词框

### 翻唱生成

支持两种模式：

- **一步翻唱**：直接传入参考音频 URL 和风格描述，快速生成
- **两步翻唱**：先提取歌词，可修改后再生成翻唱

上传方式支持 URL 或本地文件（mp3/wav/flac，6 秒 -6 分钟，最大 50MB）。

### 图片生成

- **文生图**：输入提示词直接生成
- **图生图**：上传参考图片（人物参考），保持人物一致性
- 支持 `image-01` 和 `image-01-live` 模型
- `image-01-live` 支持画风风格（漫画/元气/中世纪/水彩）和权重参数
- 批量生成 1-4 张，结果实时显示在右侧预览面板
- 多图批次支持左右轮播切换和全屏预览

### 音色克隆

上传 mp3/m4a/wav 音频（10 秒 -5 分钟，最大 20MB），创建自定义 Voice ID。

支持试听文本输入和语气词标签（如 `(laughs)`）。

### 语音合成

- 支持 50+ 预置音色，涵盖中文、英文、日文、韩文、西班牙文、法文等
- 可调节语速、音量、语调
- 支持语言增强和多声道输出
- 可生成字幕文件

### 记录管理

- 自动保存所有生成文件到本地 SQLite
- 统计各类型记录数量，点击统计卡片可筛选
- 支持在线播放/预览、下载和删除
- 翻唱记录显示歌词信息

## 环境变量

| 变量 | 说明 | 默认值 |
|------|------|--------|
| `MINIMAX_API_KEY` | 服务端兜底 API Key | 无 |
| `PORT` | 服务端口 | `5173` |
| `MINIMAX_BASE_URL` | MiniMax API 地址 | `https://api.minimaxi.com` |

## 技术栈

- **前端**：原生 HTML/CSS/JavaScript，无框架依赖
- **后端**：Node.js 原生 HTTP 服务
- **存储**：SQLite（通过 `better-sqlite3`）
- **API**：MiniMax 接口代理转发

## 注意事项

- 请勿将 API Key 提交到代码仓库
- 音色克隆需要 MiniMax 账号满足个人或企业认证要求
- 复刻他人声音前请确保拥有合法授权
